# Food Business Academy

Экспертный личный бренд Антона Коваленко — практика с 13-летним опытом в ресторанном бизнесе и доставке (владелец «Мистер Суши», Воронеж).

**Аудитория:** рестораторы, основатели доставок, повара с предпринимательскими амбициями.
**Миссия:** делиться реальным опытом без инфоцыганщины — конкретные цифры, кейсы, ошибки и рабочие решения.

## Дорожная карта (90 дней)

| Период | Фокус |
|---|---|
| **Неделя 1–2** | Бренд-документы, 10 идей, 5 скиллов, MVP content-assistant |
| **Неделя 3–4** | Первый чек-лист, lead-magnet-bot, 3 поста в TG, 1 пилотный YouTube |
| **Месяц 2** | Регулярный график: 3 поста/нед TG + 1 видео/нед YouTube + 2 шортса/нед |
| **Месяц 3** | Пилотные консультации (вручную), прототип ai-consultant |

---

## Ритм работы (5–10 ч/неделю)

1. **1 раз в неделю** — наговариваю 30–60 мин голосового по теме недели → расшифровка идёт в `knowledge-base/transcripts/`.
2. **Скилл `repurpose`** превращает её в 1 длинное видео + 3–5 шортсов + 2–3 поста.
3. **Скилл `voice-to-post`** дорабатывает каждый пост в моём тоне.
4. **Скилл `audience-question`** еженедельно — на вопросы из комментариев и DM.

Цель: 1 час моего времени = 5–7 единиц контента на 3 площадки.
