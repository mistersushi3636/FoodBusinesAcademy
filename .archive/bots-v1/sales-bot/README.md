# Sales-bot

Telegram-бот для продажи консультаций и курсов.

**Статус:** план (план в [../README.md](../README.md)).

## Когда разрабатываем

Месяц 2–3 запуска, когда:
- Появится хотя бы один продукт ([../../products/consultations.md](../../products/consultations.md))
- Будет понятная цена и формат
- Найдутся первые желающие купить (минимум 3 живые заявки руками)

## До бота

**Важный принцип:** до автоматизации — продаём вручную через личку. Это даёт:
- Понимание реальных возражений
- Калибровку цены
- Понимание формата продукта
- Кейсы и отзывы для бота

Когда руками продано 5–10 единиц — автоматизируем.

## Стек (черновик)

- aiogram + ЮKassa API (или Telegram Stars для микро-платежей)
- Notion как CRM (или Airtable)
- Calendly для бронирования слотов
- Webhook от ЮKassa → выдача доступа в закрытый канал/материалы
