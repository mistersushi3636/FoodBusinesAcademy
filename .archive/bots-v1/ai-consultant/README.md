# AI-консультант для аудитории

Telegram-бот для подписчиков: задают вопрос про ресторанный бизнес → бот отвечает голосом Антона на основе его материалов (RAG).

**Статус:** план (план в [../README.md](../README.md)).

## Когда разрабатываем

Месяц 2–3 запуска проекта, когда в `knowledge-base/` накопится 30+ материалов.

## Зачем ждать

RAG без базы знаний бесполезен. Сначала нужно:
1. Накопить расшифровки 8+ длинных видео
2. Написать 30+ постов
3. Собрать 20+ кейсов в `case-studies/`

## Стек (черновик)

- Векторная БД: Qdrant managed (бесплатный план до 1 GB)
- Embeddings: `text-embedding-3-small` (OpenAI) или `multilingual-e5-large` (локально)
- LLM: Claude Sonnet 4.6 + prompt caching на инструкции и базу
- Поиск: hybrid (BM25 + vector) для надёжности на русском

## Бизнес-модель

- Free tier: 5 вопросов в день
- Paid tier (опц.): безлимит за подписку или разовый платёж
- Логирование: все вопросы → анализ для контента (популярные темы → новые видео)
