# script-writer

Пишет сценарии видео (длинные YouTube или короткие 30–60 сек) в стиле Антона.

## Когда вызывать

- «напиши сценарий для YouTube на тему X»
- «нужен сценарий шортса про Y»
- «сделай сценарий видео из этой идеи»

## Алгоритм

1. **Прочитай контекст:**
   - `brand/tone-of-voice.md`
   - `platforms/youtube-strategy.md` (для длинных) или `platforms/shorts-strategy.md` (для коротких)
   - Если есть исходник — `knowledge-base/transcripts/...` или `content/ideas/...`
2. **Уточни** (если не сказано):
   - Формат: long (8–15 мин) или short (30–60 сек)
   - Тема и пиллар
   - Целевой портрет ЦА

### Если LONG (YouTube)

Сохрани в `content/scripts/YYYY-MM-DD-<slug>.md`:

```markdown
---
date: YYYY-MM-DD
format: youtube-long
duration: ~12 мин
pillar: <emoji>
audience: <portrait>
target_keyword: <SEO-запрос>
status: draft
---

# <Название видео>

## Превью / название (3 варианта на A/B)
1. ...
2. ...
3. ...

## Описание YouTube (для SEO)
<2–3 абзаца с ключевыми словами + ссылка на TG>

## Сценарий

### Хук (0:00–0:15)
<обещание + личный заход>

### Введение (0:15–1:00)
<кому видео, что узнаешь, почему я>

### Глава 1: <название> (1:00–3:30)
<тезисы + примеры из опыта>

### Глава 2: <название> (3:30–6:00)
...

### Глава 3: <название> (6:00–9:00)
...

### Финал и CTA (9:00–10:00)
<вывод, подписка, вопрос в коммент>

## Главы для YouTube
0:00 Хук
0:15 Введение
1:00 <Глава 1>
...

## Реквизит / показы
- Скриншоты P&L шаблона
- Фото кухни/смены
- График фудкоста
```

### Если SHORT (Shorts/Reels/TikTok)

Сохрани в `content/shorts/YYYY-MM-DD-<slug>.md`:

```markdown
---
date: YYYY-MM-DD
format: short
duration: 45 сек
pillar: <emoji>
hook_type: <шок-цифра|провал|контр-интуитив>
status: draft
---

# <Хук-заголовок>

## Кадры

**0–3 сек (хук):**
<визуал> | <текст-наложение>
*Реплика:* "..."

**3–10 сек:**
<визуал> | <текст>
*Реплика:* "..."

**10–35 сек:**
<визуал> | <текст>
*Реплика:* "..."

**35–45 сек (финал + CTA):**
<визуал>
*Реплика:* "Подпишись, разбираю общепит."

## Субтитры
<все реплики одним текстом для CapCut>

## Хэштеги
#общепит #ресторанныйбизнес #доставка
```

## Правила

- **Каждый кадр** в шортсе = одна короткая мысль на экране
- **Хук** делает обещание, которое сценарий ВЫПОЛНЯЕТ
- **Цифры из опыта Антона** — обязательно, минимум одна
- **Не использовать слова из «не говорю»** в tone-of-voice.md
