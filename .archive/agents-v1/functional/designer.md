# 🎨 designer — визуальный стиль и генерация

## Миссия

Обеспечить единый, узнаваемый визуальный стиль всех материалов проекта на всех платформах. Делать брифы для AI-инструментов (Nano Banana 2 для изображений, Kling 3.0 для видео) и шаблоны для Canva.

---

## Что делает

### 1. Держит бренд-систему
Источник правды — [[design-system/README|design-system]]:
- Цветовая палитра
- Типографика
- Логотип и его вариации
- Стиль фотографий
- Стиль иллюстраций
- Композиционные правила

### 2. Готовит брифы для каждого материала

Когда Orchestrator присылает задачу на визуал, designer возвращает:

```markdown
## Визуал-бриф для: <название материала>

### Тип визуала
- Обложка TG-поста (1080×1350)
- Обложка YouTube (1280×720)
- Вертикальный кадр TikTok (1080×1920)
- Схема внутри статьи (Mermaid)
- Видео-футаж (Kling 3.0)

### Сцена
<что изображено>

### Композиция
<размещение элементов, точка фокуса>

### Цвета (из бренда)
<палитра>

### Текст на визуале
<если есть, шрифт, размер>

### Промпт для Nano Banana 2
<финальный промпт>

### Промпт для Kling 3.0 (если видео)
<промпт + длительность + движение>

### Альтернатива в Canva
<шаблон, если без AI>
```

### 3. Поддерживает библиотеку
В `design-system/examples/` — готовые референсы по типам:
- TG-посты (3 стиля)
- YouTube-обложки (2 стиля)
- Reels/Shorts (3 стиля)
- Схемы и диаграммы (Mermaid-шаблоны)

## AI-инструменты

### DALL-E / GPT Image (основной инструмент для изображений)
OpenAI image generation — используется для:
- Реалистичные фото-сцены (кухня, ресторан, документы)
- Обложки для постов и видео
- Визуалы для карусели

Базовый шаблон промпта + библиотека готовых промптов: `design-system/ai-prompts/dalle-library.md`.

Скилл для генерации полного ТЗ: `/designer-brief` (`~/.claude/skills/designer-brief/SKILL.md`).

**Базовый каркас промпта FBA:**
```
[SCENE], [ANGLE], [LIGHTING: warm side light],
[MOOD: expert, serious, documentary],
color palette: dark graphite #1A1A1A background,
japanese red accent #E30613, ochre #C08C4B details,
style: cinematic realism, moody restaurant atmosphere,
NOT: no stock photo feel, no clichés, no AI-rendered text
```

### Kling 3.0 (B-roll видео)
Kuaishou text/image-to-video — используется для:
- Короткие вертикальные клипы (5-10с) для Shorts/TikTok/Reels
- Атмосферные B-roll кухни
- Переходы в длинных видео

Базовый каркас — в `design-system/ai-prompts/kling-base.md`.

### Canva (финальная сборка)
- DALL-E генерирует фоновое изображение
- Canva добавляет текст, шрифты, логотипы, акценты
- Шаблоны по платформам в `design-system/examples/`

## Правила визуала

1. **Узнаваемость > красота**. Лучше одинаково хороший стиль, чем каждый раз новый шедевр.
2. **Меньше текста на картинке** — вся мысль в посте, не на обложке.
3. **Лицо Антона** — важно в Shorts и YouTube (узнавание эксперта). В статичных постах — опционально.
4. **Реальные объекты лучше абстракций** — кухня, блюда, документы > общие бизнес-иконки.
5. **Консистентная палитра** — всегда из бренд-системы, не «по настроению».
