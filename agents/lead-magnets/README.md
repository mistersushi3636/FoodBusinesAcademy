# 💎 Lead Magnets — подсистема воронки подписок

> 3 субагента работают последовательно: собирают, упаковывают, производят готовый материал, который получает подписчик TG после регистрации в боте.

---

## Как работает воронка

```
[Платформа: TikTok, YT, Insta, VK...]
          ↓ CTA
[Человек переходит в Telegram → в лид-магнит-бота]
          ↓ Команда /start
[Бот предлагает выбрать магнит из 3-5]
          ↓ Пользователь выбирает
[Бот выдаёт PDF / видео / чек-лист]
          ↓
[Пользователь подписан на TG-канал]
          ↓
[Ежедневные экспертные материалы → прогрев → продажа]
```

---

## Три агента

### 1. [[agents/lead-magnets/researcher|researcher]]
**Зона:** анализ конкурентов, поиск тем, оценка спроса.
Исследует какие лид-магниты работают у других экспертов (Гребенюк, Аяз, конкурирующие рестораторы) и что находит отклик у целевой аудитории.

### 2. [[agents/lead-magnets/architect|architect]]
**Зона:** сборка концепции магнита.
Берёт темы от researcher + материалы из vault и собирает их в цельную концепцию: что будет внутри, какой формат, какая структура, какое обещание.

### 3. [[agents/lead-magnets/producer|producer]]
**Зона:** реализация в готовый файл.
Делает финальный PDF / мини-видео / чек-лист в едином дизайн-стиле (через `designer`), с обложкой, содержанием, визуалом.

## Принципы хорошего лид-магнита

1. **Один конкретный результат** — не «как построить бизнес», а «как проверить маржу за 5 минут»
2. **Быстрая польза** — читается за 5-15 минут
3. **Реальные цифры и примеры** — из опыта Мистер Суши
4. **Красивый единый дизайн** — сразу показывает уровень
5. **Явный next step** — после магнита: приглашение на консультацию / курс

## Метрики воронки

Каждый магнит отслеживается в `lead-magnets/metrics.md`:
- Сколько раз выдан (количество подписок через этот магнит)
- CTR (из рассылки TG → клик на следующий шаг)
- Конверсия в продажу (консультация/курс)

`analytics` обновляет эти метрики и корректирует какие магниты делать следующими.
