# Lead-magnet-bot

Telegram-бот, выдающий PDF-чек-лист «10 ошибок при запуске доставки» в обмен на подписку на основной канал.

**Статус:** заготовка (план в [../README.md](../README.md)).

## Когда разрабатываем

Первая итерация — неделя 3 запуска проекта (после готовых бренд-документов и первого лид-магнита).

## Что нужно перед стартом разработки

- [ ] Готовый PDF чек-листа в [../../products/lead-magnets/](../../products/lead-magnets/)
- [ ] Создан Telegram-канал
- [ ] Создан бот через @BotFather, получен токен
- [ ] Выбран хостинг (VPS / Railway / Deta)

## Файлы (когда начнём)

- `bot.py` — entry point
- `handlers/` — обработчики команд
- `db.py` — SQLite модели
- `scheduler.py` — отложенные сообщения (фолоапы)
- `requirements.txt`
- `.env.example`
- `README.md` — обновим с инструкцией деплоя
