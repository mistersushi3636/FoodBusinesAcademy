## 🎯 Ключевые принципы

1. **Узнаваемость > эстетика.** Один ход стиля, применяемый везде.
2. **Минимум текста на визуале.** Смысл в посте, картинка — якорь внимания.
3. **Реальные объекты, а не стоки.** Кухня Мистер Суши, блюда, документы.
4. **Единая палитра.** Никаких «по настроению».
5. **Антон как лицо бренда.** В видео и некоторых обложках.

## 🎨 Палитра (финализируется)

Предварительное направление — **тёмно-графитовый + акцент красный** (под ассоциацию с кухней и японской эстетикой Мистер Суши).

| Роль | Цвет | HEX | Где |
|---|---|---|---|
| Основной фон | Графит тёмный | `#1A1A1A` | Обложки, превью |
| Вторичный фон | Графит светлый | `#2D2D2D` | Блоки |
| Акцент | Японский красный | `#E30613` | Цифры, CTA, важное |
| Текст основной | Белый | `#FFFFFF` | Поверх тёмного |
| Текст приглушённый | Серый | `#A0A0A0` | Подзаголовки |
| Дерево / тёплый | Охра | `#C08C4B` | Декоративные элементы |

*Финализация — после получения обратной связи от Антона.*

## ⚡ AI-инструменты

### Nano Banana 2 — изображения
Используем для:
- Сцен с кухней / блюдами / ресторанными объектами
- Портретов (если нет живого фото)
- Ресторанных интерьеров для обложек
- Декоративных элементов PDF

Базовый промпт-каркас: [[design-system/ai-prompts/nano-banana-base|nano-banana-base]]

### Kling 3.0 — видео
Используем для:
- B-roll в TikTok / Shorts / Reels
- Атмосферные клипы для YouTube-интро
- Декоративные переходы

Базовый промпт-каркас: [[design-system/ai-prompts/kling-base|kling-base]]

## 🎭 Тон визуала

- **Экспертный, не гуру-стиль**
- **Серьёзный, но не мрачный**
- **Реальный, не стилизованный «инфобиз»**
- **Без пафоса, без криков на обложке**
- Ни одного «🚀», «💥» и «📈📈📈»
