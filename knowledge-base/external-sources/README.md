# External Sources — разбор чужих каналов

Здесь живут разборы внешних источников (Telegram-каналы, YouTube, статьи, подкасты), адаптированные под Food Business Academy через скилл `content-adapter`.

## Принцип

- Берём **суть, фреймворк, структуру** чужих экспертных материалов.
- Переписываем **с нуля своими словами**, в тоне Антона, с ресторанными кейсами и цифрами.
- Источник **не упоминаем в публикациях** (скрытая адаптация, вариант A).
- Паспорт источника и список «не-использовать-фраз» хранится внутренне — для дисциплины против плагиата.

## Структура каждой подпапки

```
<slug>/
├── README.md           # паспорт источника + сводка разбора
├── raw/                # сырые выгрузки (посты, транскрипты, скриншоты)
├── adaptations.md      # карточки адаптации идей
├── output/             # готовые материалы (посты, шортсы, сценарии)
└── plagiarism-check.md # фразы источника, которые нельзя использовать
```

## Как запускать

В Claude: «разбери канал @xyz» / «адаптируй источник <url>» — запускается скилл `content-adapter`.
