## Ключевые фреймворки автора (те, что могут быть полезны)

1. **Система > звёзды** — сила отдела не в героях, а в процессе (скрипты, КПИ, автоматизация, аналитика).
2. **4 уровня освоения профессии** — некомпетентность → компетентность → мастерство → гениальность. Предприниматель должен работать из четвёртого уровня.
3. **«Зона гениальности» vs профессионализм-ловушка** — опасно застрять в «хорошо делаю много разного», нужно фокусироваться на одном ядре.
4. **Скорость vs точность** — хаотичная быстрота создаёт плато, точная методичная работа пробивает его (метафора укладки плитки).
5. **Минимальный подтверждающий результат** (у автора — «точка ноль») — первый жизнеспособный срез, который доказывает работоспособность модели.
6. **СП-схема для производственников** — гений в создании продукта отдаёт продажи партнёру по франшизе 50/50, масштабируется в регионах.
7. **Люди-продукты / люди-системы** — производственный гений не умеет продавать; ему нужен коммерческий партнёр.
