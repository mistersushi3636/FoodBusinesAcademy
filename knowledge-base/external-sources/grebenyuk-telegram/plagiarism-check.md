# Антиплагиат: фразы и термины, которые НЕ используем

> Правило: суть сохраняем, формулировки — свои. Ни одна авторская фраза-ярлык ниже не должна попасть в наш материал дословно или с минимальной перестановкой.

## Термины-ярлыки автора → наши замены

| Термин автора | Наша замена (варианты) |
|---|---|
| «Зона гениальности» | ядровая сила · то, что делаешь лучше всех · коренная способность · твоё единственное преимущество |
| «Скорость vs быстрота» | хаос vs точность · быстро-криво vs быстро-чисто · суета vs методичная скорость |
| «Точка ноль» | минимально работающая модель · базовый жизнеспособный срез · зелёная зона · первый уверенный уровень |
| «Люди-продукты» / «люди-системы» | мастер цеха vs бизнес-оператор · руки vs голова · производственник vs управленец |
| «Отдел продаж по захвату рынка» | — (вообще не использовать формулировку) |
| «Гениальные скрипты продаж» | рабочие заготовки для общения · формулы ответов · готовые связки для разговора |
| «Резалтинг» | — (не использовать) |
| «Система, а не звёзды» | выстроенный процесс вместо одного героя · поточная работа вместо ставки на «суперлюдей» |

## Структурные рамки автора → наши

| Рамка автора | Наша версия |
|---|---|
| «4 уровня: некомпетентность → компетентность → мастерство → гениальность» | можно использовать логику перехода (новичок → уверенный → эксперт → мастер), но **переименовать уровни** и добавить ресторанные маркеры |
| «СП 50/50 с коммерческим» | партнёрство повара с управляющим · развод по функциям (кухня vs зал/деньги) — описать своими словами на примере общепита |
| Метафора плитки → точность | **заменить метафору**: нарезка рыбы, кладка ингредиентов в ролл, расстановка по холодильникам |

## Проверочный чек после написания

1. Открываешь готовый пост.
2. Ищешь в нём любую фразу из таблицы выше → если нашёл, переписываешь.
3. Ищешь фрагменты из 5+ слов подряд, которые могли бы быть в источнике → перефразируешь.
4. Проверяешь, что все метафоры — ресторанные (кухня/зал/доставка), а не из отдела продаж.
5. Убеждаешься, что автор нигде не упоминается по имени.

## Лексика, которую всегда используем вместо «продажной»

- «гость» вместо «клиент» (где уместно в зале)
- «смена», «кухня», «линия» — ресторанные слова
- «заказ», «чек», «позиция в меню» — из операционки
- «агрегатор», «самовывоз», «доставка» — из доставки
