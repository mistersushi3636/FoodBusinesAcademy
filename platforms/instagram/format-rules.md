# Instagram — правила формата

> **Режим публикации: SEMI-AUTO.** Бот готовит draft + картинку → шлёт Антону в личку → Антон публикует руками.
> Причина: официальный Graph API требует Meta review (~2-4 нед), unofficial API банит аккаунты.

## Что бот формирует

1. **Caption** (текст в посте)
2. **Hook line** (первые 125 символов — видно до «more»)
3. **Hashtags** (10-20)
4. **Image** (1080×1080 или 1080×1350, 4:5)
5. **Storyline** для Reels (если short-video формат)

## Caption: длина
- Максимум 2 200 символов
- Реалистичная цель: 600–1 100 символов
- Hook на первой строке (125 знаков) — критично

## Структура caption
```
Hook                (≤125 знаков, цепляет до «...more»)
Контекст            (2-3 строки)
Конкретика          (3-5 строк, цифры/шаги)
CTA                 (1 строка) — «Подробности в TG → ссылка в био»
Хэштеги             (отдельный блок внизу)
```

## Hook-формулы (рабочие в IG)
- «Я слил Х рублей на Y. Вот что понял»
- «3 вещи, которых нет в учебниках по [теме]»
- «Если бы открывал заново — изменил бы это»
- Конкретная цифра в первом предложении

## Хэштеги
- 10-20 штук, в конце или в первом комменте
- Mix: широкие (#ресторанныйбизнес) + узкие (#доставкасуши) + брендовые (#mistersushi)
- Заранее заготовленный набор: `design-system/ig-hashtags.md`

## Reels — отдельные правила
- 15–60 сек
- Subtitles ОБЯЗАТЕЛЬНЫ (60% смотрят без звука)
- Hook в первые 1.5 сек
- Текст на экране — крупный, белый с чёрной обводкой
- Pace: смена кадра каждые 2-3 сек

## Картинка для feed-поста
- 1080×1350 (4:5) — больше места в ленте
- DALL-E 3 → quality=hd, size=1024×1792 → ресайз до 1080×1350
- Стиль из `design-system/ai-prompts/`

## CTA на TG
- Не давать прямую ссылку (IG режет охват за внешние линки)
- В био указать «t.me/Food_Busines_Academy»
- В caption: «Полная версия — в TG (ссылка в био)»

## Время публикации
- Будни 12:00 или 19:00 МСК
- Reels — четверг/воскресенье вечер (статистически лучше)

## Что бот шлёт Антону в ЛС

```
📸 IG draft готов.

[caption preview]

🖼 Картинка: <attached>
🏷 Hashtags: <list>
📐 Соотношение: 4:5

Опубликовать → IG App → New Post → выбрать картинку → вставить caption → опубликовать.
```
