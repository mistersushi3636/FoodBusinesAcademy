# YouTube — правила формата

> **Режим публикации: SEMI-AUTO.** Бот готовит сценарий + thumbnail-промпт + описание → шлёт Антону в ЛС → Антон загружает руками.
> Причина: автоматический upload через Data API возможен, но требует OAuth + refresh tokens. Рассмотрим во второй итерации, если объём публикаций оправдает.

## Два формата

### Shorts (60 сек)
- Vertical 9:16 (1080×1920)
- Hook в первую секунду
- Subtitles обязательны
- Music из YT library (без авторских прав)

### Long video (5-30 мин)
- Horizontal 16:9 (1920×1080)
- Структура с timestamps
- Thumbnail — отдельная задача, обязателен

## Что бот формирует

| Формат | Output |
|---|---|
| Shorts | сценарий 60 сек (хук + 3 точки + CTA) + thumbnail-промпт |
| Long | outline + ключевые цитаты + thumbnail-промпт + описание + теги + chapters |

## Сценарий Shorts
```
0:00 - 0:01  HOOK (одна цепляющая фраза)
0:01 - 0:50  POINTS (3 пункта по 15 сек)
0:50 - 0:60  CTA («Подписывайся на TG — ссылка в описании»)
```

## Outline long video
```
0:00 Hook + intro (45 сек)
0:45 Что узнаешь (chapters preview)
01:30 Block 1: [topic]
05:00 Block 2: [topic]
...
N:NN Резюме + CTA на TG-канал
```

## Title формула (long)
- ≤60 символов
- Конкретное число + результат: «Открыл доставку за 90 дней. Вот 3 главные ошибки»
- Без clickbait

## Title формула (Shorts)
- ≤40 символов
- Хук + obvious value: «3 цифры перед открытием доставки»

## Description
- Первые 150 символов критичны (видно в snippet)
- Структура:
```
[1 предложение что в видео]

📌 Ссылки:
- TG-канал: t.me/Food_Busines_Academy
- Чек-лист «10 ошибок»: @fba_leadmagnet_bot

🕐 Главы:
00:00 ...
05:00 ...

🏷 #ресторанныйбизнес #доставка ...
```

## Tags
- 5-15 тегов
- Mix: широкие + узкие + бренд

## Thumbnail (для long)
- 1280×720
- Лицо Антона + 1 крупная фраза (3-5 слов)
- Контраст: красный/жёлтый/белый
- Не больше 2 цветовых акцентов
- DALL-E генерит фон, лицо Антона — реальное фото вкладывается через Canva

## CTA на TG
- В пинне комментария: «Полная инструкция в TG: t.me/Food_Busines_Academy»
- В описании: первой ссылкой
- В видео голосом: «Все цифры и шаблоны — в моём TG, ссылка в описании»

## Что бот шлёт Антону в ЛС

```
📺 YouTube {Shorts|Long} draft готов.

🎬 Сценарий: <text>
🏷 Title: <text>
📝 Description: <text>
🖼 Thumbnail prompt: <text>
🎯 Теги: <list>

Загрузить → YT Studio → Upload → вставить.
```
