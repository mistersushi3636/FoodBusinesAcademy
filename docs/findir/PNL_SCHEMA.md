## Доходы
**Ввод: ежедневно через бот / форму `Ввод за день`.**

| Поле | Описание |
|------|----------|
| `revenue_delivery` | Выручка доставки ₽ |
| `revenue_pickup` | Выручка самовывоза ₽ |
| `orders_delivery` | Заказов доставки |
| `orders_pickup` | Заказов самовывоза |

**Расчётные:**
- Средний чек = выручка / заказы (для доставки, самовывоза, общий)
- Доля доставки/самовывоза = % от общей выручки

## Себестоимость
**Ввод: раз в месяц.**

| Категория | Описание |
|-----------|----------|
| `food_supply` | Затраты на продукты у поставщиков |

**Food cost % = food_supply / общая выручка × 100.**

## ФОТ
**Ввод: 2 раза в месяц или вручную.**

**Категории сотрудников:**
- `managing` — Управляющий
- `sushef` — Су-шеф
- `chef` — Повар
- `admin` — Администратор
- `courier` — Водитель-курьер
- `chef_trainee` — Повар-стажёр
- `admin_trainee` — Администратор-стажёр
- `accounting` — Бухгалтерия (общая)
- `marketing` — Маркетинг (общая)

**Скрыто от роли `manager`:** `managing`, `accounting`, `marketing`.

## Итоги (расчётные)

```
EBITDA = выручка − себестоимость(food_supply) − опер.расходы
ЧП = EBITDA − налоги − кредиты(bank_credit)
ЧП % = ЧП / выручка × 100
```

**Опер. расходы =** ежедневные(магазин+хоз+еда персонала) + ФОТ + интернет + оборудование + банк.комиссии + коммуналка/аренда + маркетинг + сайт + прочее.

## План/факт
- Месячный план по любой категории — ввод вручную в `/plan-fact`.
- Δ ₽ = факт − план; Δ % = Δ / план × 100.
- Дополнительно — среднее по дням недели за 3 месяца для сравнения.
