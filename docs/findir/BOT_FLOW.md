# Бот — флоу взаимодействия

## Авторизация
Только TG IDs из `.env` (`OWNER_TG_ID`, `MANAGER_TG_ID`). Все остальные получают отказ + видят свой ID для передачи Антону.

## Главное меню (`/start`)
Reply-клавиатура:

| Владелец | Управляющий |
|----------|------------|
| 💰 Внести выручку | 💰 Внести выручку |
| 🛒 Расходы за день | 🛒 Расходы за день |
| 📅 Месячный расход | — |
| 👥 Выплата ФОТ | — |
| 📊 Сводка | 📊 Сводка |
| ℹ️ Помощь | ℹ️ Помощь |

## Команды
- `/start` — меню
- `/daily` — ввести выручку + ежедневные расходы
- `/monthly` — внести месячный расход (только владелец)
- `/summary` — сводка по текущему месяцу
- `/help` — список команд

## Флоу: `/daily` (ввод за день)
1. Дата (по умолчанию сегодня)
2. Выбор филиала (inline-кнопки: Ростовская / 9 Января)
3. Выручка доставка ₽
4. Выручка самовывоз ₽
5. Заказов доставка
6. Заказов самовывоз
7. Покупки магазин ₽
8. Хоз/тара/упаковка ₽
9. Еда персонала ₽
10. Подтверждение → POST `/api/daily` → ✅

## Флоу: `/monthly` (только владелец)
1. Период (текущий месяц)
2. Филиал (Общие / Ростовская / 9 Января)
3. Категория (15 inline-кнопок)
4. Сумма ₽
5. Заметка (или `-`)
6. POST `/api/monthly` → ✅

## Флоу: `/summary`
- Запрос `GET /api/summary?period=YYYY-MM`
- Управляющий видит: выручка, заказы, ср.чек, food cost
- Владелец дополнительно: ФОТ %, EBITDA, ЧП ₽ + %

## API контракт (бот ↔ дашборд)
**Аутентификация:** `Authorization: Bearer ${DASHBOARD_API_KEY}`

| Endpoint | Метод | Назначение |
|----------|-------|-----------|
| `/api/daily` | POST | Выручка + ежедневные расходы |
| `/api/monthly` | POST | Месячный расход (owner only) |
| `/api/summary` | GET | KPI за период |

## Будущее (не в MVP)

- Алерты при превышении бюджета по категории
- Импорт CSV из Goulash.tech (при открытии API)
