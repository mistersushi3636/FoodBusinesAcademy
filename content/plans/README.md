# 📅 Content Plans — еженедельные контент-планы

## Откуда берутся

Автоматически создаются `content-planner` скиллом каждый понедельник в 11:00.

Формат имени: `YYYY-WW-plan.md` (ISO неделя).

---

## Структура файла

См. шаблон в `agents/functional/content-marketer.md` → раздел «Формат 30-дневного плана».

---

## Статусы плана

| Статус | Значение |
|---|---|
| `draft` | Только что сгенерирован, ждёт ответ Антона |
| `review` | Антон дал правки, content-marketer корректирует |
| `approved` | Утверждён, передан calendar и platform-агентам |
| `archived` | План старше 30 дней, сохранён для истории |

---

## История планов

Все старые планы остаются в папке. content-marketer читает последние 4 плана для трекинга решений и паттернов.
