# Редакционный календарь

Шаблон еженедельного производства контента под бюджет 5–10 ч/неделю.

## Недельный ритм (типовая неделя)

| День | Активность | Время |
|---|---|---|
| **Пн** | Запись 1 длинного голосового (30–60 мин) по теме недели | 1 ч |
| **Пн вечер** | Расшифровка → `knowledge-base/transcripts/`. Запуск скилла `repurpose` → черновики 1 видео + 3–5 шортсов + 2–3 поста | 1 ч |
| **Вт** | Доработка длинного видео (сценарий, съёмка) | 2 ч |
| **Ср** | Монтаж длинного видео (или делегирую) + публикация поста №1 в TG | 1.5 ч |
| **Чт** | Публикация шортса №1 + поста №2 | 0.5 ч |
| **Пт** | Публикация поста №3 (история/кейс) + шортса №2 | 0.5 ч |
| **Сб** | Скилл `audience-question` → ответы на DM/комменты, лёгкий пост | 1 ч |
| **Вс** | Выходной / планирование темы следующей недели | 0.5 ч |

**Итого:** ~8 часов/неделю.

## Что выходит за неделю

- 1 длинное YouTube-видео (10–20 мин)
- 2–3 шортса (60 сек)
- 3 поста в Telegram
- 1 ответ на вопрос подписчика (отдельный пост)

= **6–7 единиц контента из 1 голосового сырья.**

## Распределение по пилларам (ротация)

Каждую неделю — фокус на 1 главный пиллар. По кругу:

| Неделя | Главный пиллар | Тема (пример) |
|---|---|---|
| 1 | 💰 Деньги | Сколько стоит запустить доставку в 2026 |
| 2 | ⚙️ Операционка | График смен, который я использую |
| 3 | 📣 Маркетинг | Яндекс.Еда vs свой сайт |
| 4 | 📖 Кейсы | Самая дорогая ошибка моих 3 лет |
| 5 | 🔮 Тренды | Dark kitchen — заходить или нет |
| 6 | 💰 Деньги | Считаем фудкост за 10 минут |
| ... | (повтор) | ... |

## Шаблон недельного плана

Создай файл `content/calendar/2026-WW.md` (где WW — номер недели) по шаблону:

```markdown
# Неделя WW — Тема: <название>
Пиллар: <💰/⚙️/📣/📖/🔮>
Боль ЦА: <какую боль закрываем, какой портрет>

## Сырьё
- Голосовое: <ссылка/файл>
- Расшифровка: knowledge-base/transcripts/YYYY-MM-DD-tema.md

## План публикаций
- [ ] YouTube long: <название> | срок: Ср
- [ ] Шортс 1: <хук> | срок: Чт
- [ ] Шортс 2: <хук> | срок: Пт
- [ ] TG пост 1: <тема> | срок: Ср
- [ ] TG пост 2: <тема> | срок: Чт
- [ ] TG пост 3 (кейс): <тема> | срок: Пт

## Метрики после
- Просмотры YT:
- Просмотры шортсов:
- Подписки за неделю:
- Лучший пост (по реакциям):
```

## Идеи на 4 первых недели (для старта)

1. **Неделя 1:** «Сколько реально стоит открыть доставку суши в 2026: разбор сметы»
2. **Неделя 2:** «Чек-лист открытия и закрытия смены, который у меня работает 5 лет»
3. **Неделя 3:** «Яндекс.Еда vs свой сайт — где больше прибыли (моя экономика)»
4. **Неделя 4:** «Как я начинал поваром и стал владельцем — история без розовых очков»
